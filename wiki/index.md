Here’s the guide in plain prose.

To edit an existing page, open it and start typing in the editor; your changes are saved automatically as you go. If you need a new page, use the “New file path” field (for example, enter `notes/ideas`), submit it, and then write your content in the editor that appears.

Below are the key bits of Markdown you can use while writing, each shown with the raw input and the rendered result.

Headings  
Raw:
```markdown
# Title One
## Title Two
### Title Three
```
Rendered:
```text
Title One
Title Two
Title Three
```

Bold and italic  
Raw:
```markdown
This is **bold** text and this is *italic* text.
```
Rendered:
```text
This is bold text and this is italic text.
```

Lists  
Raw:
```markdown
- First thing
- Second thing
  - Nested item
1. Numbered item
2. Another numbered item
```
Rendered:
```text
• First thing
• Second thing
  • Nested item
1. Numbered item
2. Another numbered item
```

Links to other pages  
Raw:
```markdown
See the [home page](/) or jump to [Ideas](/notes/ideas).
```
Rendered:
```text
See the home page or jump to Ideas.
```

That’s all you need to write and create pages in Aenyrathia.
